# MINI-CALCULATOR
The Mini Calculator is a simple Python-based project that performs basic arithmetic operations such as Addition, Subtraction, Multiplication, and Division.
This project is designed for beginners to understand Python fundamentals, user input handling, conditional statements, and basic logic building.
